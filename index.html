Hello-ref test page

This page is meant to help demonstrate how hello-ref work.

What it does basically is grabs the value of ?Ref= from the URL and presents a custom message.

You get to control the design and text for each referral.


Main use case

When you share your website on a network, append ?ref= to the URL

so you can present a message when people hop in

Check the gitHub repo for an updates release notes
